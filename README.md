# Terraform Learning (A2–B1)

This repository is my place to learn Terraform.  
You will find short notes, small examples, and a PDF cheat sheet.

## Goals
- Learn the Terraform basics.
- Create small, clear examples.
- Use simple step-by-step guides.
- Keep code easy to read.

## What is inside
- `notes/` – short notes in simple English.
- `examples/` – small .tf examples to practice.
- `Terraform_Cheat_Sheet.pdf` – quick reference.

## Requirements
- Terraform (latest stable)
- A terminal and a code editor

## How to run an example
```bash
cd examples/01-hello
terraform init
terraform plan
terraform apply
